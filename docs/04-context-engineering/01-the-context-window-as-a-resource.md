# The Context Window as a Scarce Resource

## The Budget Metaphor
- Every token has a cost (latency, money, attention)
- The context window is not infinite attention—it's a budget to manage

## The 3-Zone Model
- Zone 1: System Prompt (fixed cost, high leverage)
- Zone 2: Retrieved Context (variable cost, medium leverage)
- Zone 3: Conversation History (growing cost, diminishing returns)

## Allocation Strategies
- Token budgeting by priority
- When to summarize vs. when to truncate
- The "needle in a haystack" problem and mitigation
